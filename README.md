## 3.模板介绍

1. `index1.html`为自定义通知邮件模板
2. `index.html`为自定义博主通知邮件主题
3. `textarea`文件中自定义回复是以textarea标签写的，兼容性较差

## 4.模板使用

1. 此邮箱模块试用于【[twikoo](https://twikoo.js.org/)】等等
2. 修改`变量`可适配其他的评论框架
